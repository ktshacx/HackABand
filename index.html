You ship art, We ship Wrist Band

Elasic woven wrist bands with NFC chip, with art on both front and back.

RSVP Slack

What to do ?

  • Draw a rough art on paper
  • Use any ui/art tool like Figma or Procreate
  • Draw both front and back (180 x 20) px
  • Make a PR on github with a folder with your name containing front.png and back.png
  • Fill this form after PR is accepted.

FAQs

HackABand is a YSWS organised in/by Hack Club, in which you can design your own Wrist Band.

To participate, draw your art, use any UI/art tool, and make a PR on GitHub with your art files.

You can use any UI/art tool like Figma or Procreate to create your art.

It will run for 1 months (if possible we may extend it).
